# Лабораторная работа №4
# Библиотека книг с индексацией и симуляцией

## Описание проекта

Проект представляет собой систему управления библиотекой книг с поддержкой индексации для быстрого поиска и псевдослучайной симуляции работы библиотеки. Система позволяет:

- Хранить коллекцию книг с метаданными (название, автор, год, жанр, ISBN)
- Индексировать книги по ISBN, автору и году издания для быстрого поиска
- Выполнять поиск книг по различным критериям
- Запускать псевдослучайную симуляцию работы библиотеки с различными событиями

## Установка и настройка

### Требования

- Python 3.12 или выше
- pip или uv (менеджер пакетов)

### Установка зависимостей

```bash
pip install -r requirements.txt
```

## Запуск программы

### Основной способ запуска

```bash
python3 -m src.main
```

Программа запросит:
1. Количество шагов симуляции (по умолчанию 20)
2. Seed для генератора случайных чисел (опционально, для воспроизводимости)

### Пример запуска

```bash
$ python3 -m src.main
Введите количество шагов симуляции (по умолчанию 20): 10
Введите seed для генератора случайных чисел (опционально, Enter для пропуска): 42

[СИМУЛЯЦИЯ] Начало симуляции библиотеки
[СИМУЛЯЦИЯ] Количество шагов: 10
[СИМУЛЯЦИЯ] Seed: 42
[Шаг   1] Событие: add_book
Добавлена книга: 'Война и мир' автора Лев Толстой
...
```


**Пример использования:**
```python
from src.simulation import run_simulation

# Запуск симуляции с 10 шагами
run_simulation(steps=10)

# Запуск симуляции с seed для воспроизводимости
run_simulation(steps=20, seed=42)
```
